# Chapter 01 - Variables

## 📖 What are Variables?

A variable is a named storage location used to store data.

In Go, variables allow us to store values that can be changed while the program is running.

---

## 💡 Syntax

```go
var name string
```

or

```go
name := "Negar"
```

---

## 🧠 What I Learned

- How to declare a variable
- Variable types
- Short variable declaration (`:=`)
- Updating variable values

## ❌ Common Mistakes

Using `=` instead of `:=` when declaring a new variable.

Example:

```go
name = "Negar"
```

This causes an error if `name` has not been declared.

---

## 📌 Summary

Variables are used to store data that can be reused or changed during program execution.
