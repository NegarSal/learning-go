# Exercises

## Exercise 1

Create a program that asks the user for a service name.

Use a `switch` statement.

If the service is:

- nginx → print "Web Server"
- mysql → print "Database Server"
- redis → print "Cache Server"

Otherwise print:

```text
Unknown service.
```

## Exercise 2

Create a program that asks the user for a container status.

Use a `switch` statement.

If the status is:

- running → print "Container is running."
- stopped → print "Container is stopped."
- paused → print "Container is paused."

Otherwise print:

```text
Unknown status.
```
